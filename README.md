## Ejecución del programa 🚀

### Windows 

_Descargar y ejecutar_

```diff
EN DESARROLLO
```

### Distribución de Linux basada en Debian 

_Pasos a seguir (comandos de terminal):_

```
git clone https://github.com/Alexvidalcor/SuiteMinigames
cd SuiteMinigames/
python3 src/install.py
./execute.sh
```
